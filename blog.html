What is the Document Object Modal (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, representing a webpage's structure as a tree of objects. It allows developers to dynamically access, modify, and manipulate HTML or XML content, including elements, attributes, and styles, using scripts like JavaScript. Through the DOM, changes can be made to the page in response to user interactions, without requiring a full page reload, enabling dynamic and interactive web experiences. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events for its child elements. Instead of adding individual listeners to each child, the event "bubbles up" to the parent, which can capture and process the event. This is useful because it improves performance, reduces memory usage, and simplifies code by handling dynamic elements that may be added or removed after the page loads.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes in the DOM, you can use:
1. setAttribute() to set or update an attribute: element.setAttribute('attribute', 'value');
2. getAttribute() to retrieve an attribute's value: let value = element.getAttribute('attribute');
3. removeAttribute() to remove an attribute: element.removeAttribute('attribute');
